Entorno QVD Server
~~~~~~~~~~~~~~~~~~

Ubuntu Linux
^^^^^^^^^^^^
El entorno QVD Server está diseñado para ser instalado en GNU/Linux Ubuntu 10.04 (Lucid Lynx). A pesar de que podríamos proporcionar el tarball del código fuente preferimos que simplemente uses los paquetes que proporcionamos para instalar y configurar tu entorno, ya que esto ayudará a asegurar que todas las dependencias necesarias están disponibles también.

Los paquetes para el entorno del QVD server están disponibles en <a href="https://launchpad.net/~qvd-qindel/+archive/qvd-3.0">https://launchpad.net/~qvd-qindel/+archive/qvd-3.0</a>.

Ejecuta el siguiente comando para actualizar `/etc/apt/sources.list` en cualquier sistema donde quieras instalar los componentes QVD

----
sudo apt-add-repository ppa:qvd-qindel/qvd-3.0
----

[NOTA]
===================================================================
Si estás usando Lucid Lynx, y no lo has echo ya, quizás necesites instalar *python-software-properties* antes de intentar usar *apt-add-repository*.
===================================================================

Ejecuta `sudo apt-get update` después de añadir el repositorio de QVD.

El repositorio QVD ofrece los siguientes paquetes:

* *qvd-demo-single-instance-nosupport*: meta-paquete para instalar una configuracion de demostración en un sólo equipo
* *qvd-client*: Cliente GUI de QVD
* *qvd-l7r*: Componente de intermediación de enrutamiento de capa 7 de QVD Server
* *qvd-hkd*: Demonio de limpieza para QVD Server
* *qvd-admin*: Herramientas de linea de comandos para gestionar usuarios, máquinas virtuales e imagenes de disco de máquinas virtuales
* *qvd-vma*: Agente que corre dentro de las máquinas virtuales (Virtual Machine Agent)
* *qvd-wat*: Herramientas Web de Administracion (Web Administration Tool)
* *qvd-db*: Base de datos central para la plataforma

Por favor, consulta la documentación para obtener instrucciones sobre cómo instalar y construir una solución QVD.

SuSE Linux
^^^^^^^^^^
El entorno de QVD Server se puede instalar también en SuSE Linux
Enterprise Server 11 (SP1 x86_64).

Los paquetes para el entorno de QVD Server están disponibles en <a href="https://build.opensuse.org/project/show?project=home%3Aqvd%3Aqvd-3.0">
https://build.opensuse.org/project/show?project=home%3Aqvd%3Aqvd-3.0</a>.

Ejecuta el siguiente comando para añadir el repositorio a tu sistema.

----
sudo zypper ar-3.0.0 http://download.opensuse.org/repositories/home:/qvd:/qvd-3.0/SLE_11_SP1 QVD
----
Ejecuta `sudo zypper refresh` después de añadir el repositorio de QVD. Tendrás que aceptar el certificado que se te presenta. Recomendamos que aceptes el certificado permanentemente pulsando 'a' en tu teclado.

Puede que te pida el disco de instalación para completar la operación.

Por favor, consulta la documentación para obtener instrucciones sobre cómo instalar y construir una solución QVD.


Cliente GUI de QVD
~~~~~~~~~~~~~~~~~~
El Cliente GUI de QVD puede instalarse en cualquier máquina que vaya a necesitar acceder a un escritorio QVD.  Los paquetes de instalación están disponibles para Linux y para Windows.

Linux (Ubuntu)
^^^^^^^^^^^^^^
En sistemas operativos basados ​​en Ubuntu Linux, sólo tienes que añadir el PPA de QVD y hacer un `apt-get install qvd-client`. El cliente debería aparecer dentro del menú de aplicaciones de tu escritorio.
La PPA se puede encontrar en: <a href="https://launchpad.net/~qvd-qindel/+archive/qvd-3.0">https://launchpad.net/~qvd-qindel/+archive/qvd-3.0</a>

Linux (SuSE)
^^^^^^^^^^^^
Para usuarios de SuSE y openSUSE, es posible descargar nuestro paquete de cliente directamente desde el Build Service de openSUSE, en: <a href="https://build.opensuse.org/package/show?package=qvd-client&project=home%3Aqvd%3Aqvd-3.0">https://build.opensuse.org/package/show?package=qvd-client&project=home%3Aqvd%3Aqvd-3.0</a>

Windows
^^^^^^^
El instalador de cliente GUI de QVD para Windows está disponible en el siguiente enlace:

* link:http://theqvd.com/downloads/windows/qvd-client-3.0.0-1-setup.exe[Cliente GUI QVD para Windows]

El instalador de Windows debería instalar el cliente en las últimas versiones de Microsoft Windows.

Discos de imágenes de demo QVD
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Si estás probando QVD por primera vez, y no quieres tener que crear tu propia imagen de disco para correr dentro de tu Máquina Virtual QVD, proporcionamos algunas imágenes sencillas de demo para que puedas usar.

Puedes descargar una de las siguientes Imágenes de disco (DI) de demotración DI de los siguientes enlaces:

*   *link:http://theqvd.com/downloads/demo/demo-ubuntu-gnome.img.gz[QVD Demonstration DI (Ubuntu 10.10 – Escritorio Gnome) [740Mb]]*

[NOTA]
========================================================================================
Esta imagen de demo es una instalación muy básica de una Ubuntu 10.10 (Maverick Meerkat) e incluye el navegador Firefox (con Adobe Flash Player instalado), El cliente de correo Evolution, y OpenOffice.org.
========================================================================================

*   *link:http://theqvd.com/downloads/demo/demo-ubuntu-kde.img.gz[QVD Demonstration DI (Ubuntu 10.10 – Escritorio KDE) [717Mb]]*

[NOTA]
============================================================================================
Esta imagen de demo es una instalación muy básica de una Ubuntu 10.10 (Maverick Meerkat) with KDE
e incluye el navegador web Konqueror (con Adobe Flash Player instalado), Kontact, y LibreOffice.
============================================================================================

Estas imágenes simplemente tienen la intención de demostrar cómo QVD es capaz de servir a un escritorio virtual a un usuario. Es posible modificar una imagen para satisfacer tus propias necesidades una vez que lo hayas descargado, pero te recomendamos crear tu propia Imagen de disco (ID) desde cero si estás creando un entorno a nivel de producción, o considerar usar nuestra Commercial Edition o Cloud Edition y conseguir soporte profesional para la creación de las imágenes que necesites.

Ten en cuenta que los archivos DI que facilitamos están comprimidas. Si tu navegador no es capaz de manejar esto automáticamente por ti, puede ser que tengas que descomprimir el fichero manualmente tu mismo una vez que hayas terminado la descarga:

----
gunzip qvd-demo-gnome.img.gz
----
