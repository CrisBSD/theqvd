You are able to specify the following options to control the node daemon:

----
noded.as_user = root
noded.pid_file = ${path.run}/hkd.pid
----

The values set above are the default values.

* *noded.as_user*: the user that should be used to run the QVD Node daemon
* *noded.pid_file*: the path to the PID file that is created when the daemon is running
